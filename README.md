# پچ RTL برای Claude Desktop

**پشتیبانی کامل از فارسی و عربی (راست‌به‌چپ) برای Claude Desktop در ویندوز و macOS.**

متن RTL را به‌صورت خودکار شناسایی می‌کند، جهت را در لحظه تنظیم می‌کند و متن فارسی را با فونت وزیرمتن نمایش می‌دهد — بدون اینکه متن انگلیسی یا بلوک‌های کد خراب شوند.

## ✨ قابلیت‌ها

- **شناسایی خودکار متن RTL** در پاسخ‌های Claude و کادر ورودی، با تنظیم جهت در لحظه — حتی در حین دریافت پاسخ.
- **نمایش متن فارسی با فونت وزیرمتن** — فونت متغیر و مدرن طراحی‌شده برای فارسی و عربی، به‌صورت data URI تعبیه‌شده (بدون نیاز به اینترنت).
- **مدیریت کد، فرمول‌های ریاضی و جداول** — بلوک‌های کد و LaTeX (مثل `$x^2$`) در محیط RTL همچنان چپ‌به‌راست می‌مانند، و جداول فارسی از راست به چپ نمایش داده می‌شوند.
- **رفع باگ پنجره در سیستم‌های RTL** (ویندوز): وقتی زبان ویندوز فارسی یا عربی است، رابط کاربری Claude Desktop به‌طور اشتباه RTL می‌شود. این پچ آن را برطرف می‌کند.
- **پشتیبان‌گیری از همه فایل‌ها** قبل از هر تغییر، با بازگشت خودکار در صورت بروز خطا.
- **ماندگاری پس از به‌روزرسانی** (اختیاری)، از طریق میانبر دسکتاپ یا به‌روزرسانی خودکار.

## 🚀 نصب سریع — Windows

سرویس **Windows PowerShell** (آبی‌رنگ) را باز کنید و اجرا کنید:

```powershell
irm https://raw.githubusercontent.com/mhghotbi/claude-desktop-rtl-patch/main/install.ps1 | iex
```

پنجره UAC ظاهر می‌شود. روی **Yes** کلیک کنید تا دسترسی مدیر داده شود، سپس از منو گزینه **۱. نصب** را انتخاب کنید.

---

## 🍎 نصب سریع — macOS

نسخه مک یک **Claude-RTL.app** جداگانه در `~/Applications` می‌سازد — برنامه اصلی دست‌نخورده باقی می‌ماند.

**نیازمندی‌ها:** Node.js (از [nodejs.org](https://nodejs.org/)) و Xcode Command Line Tools (`xcode-select --install`).

```bash
git clone https://github.com/mhghotbi/claude-desktop-rtl-patch
cd claude-desktop-rtl-patch
bash patch-mac.sh
```

گزینه **۱. نصب** را انتخاب کنید. سپس **Claude RTL** را از `~/Applications` باز کنید.

## 🔄 ماندگاری پچ پس از به‌روزرسانی (ویندوز)

به‌روزرسانی‌های Claude Desktop پچ را پاک می‌کنند. دو راه‌حل:

- **میانبر به‌روزرسانی سریع (گزینه ۳):** میانبر دسکتاپی به نام **"Update Claude RTL"**. با دوبار کلیک، پچ را مجدداً اعمال می‌کند.
- **به‌روزرسانی خودکار (گزینه ۴):** یک تسک زمان‌بندی‌شده سبک که وقتی نسخه جدید `claude.exe` راه‌اندازی می‌شود، پچ را دوباره اعمال می‌کند.

در macOS، کافی است پس از هر به‌روزرسانی `bash patch-mac.sh` را دوباره اجرا کنید.

## 🗑️ حذف

**سیستم عامل Windows:** اسکریپت را اجرا کنید و گزینه **۲. بازگشت به حالت اصلی** را انتخاب کنید. اگر به‌روزرسانی خودکار را فعال کرده‌اید، ابتدا گزینه **۵** را اجرا کنید.

**سیستم عامل macOS:** `bash patch-mac.sh restore` را اجرا کنید (یا گزینه ۲ از منو). Claude.app اصلی دست‌نخورده است.

## 🛠️ عیب‌یابی

ایراد **"Node.js (npx) is required":** Node.js را از [nodejs.org](https://nodejs.org/) نصب کرده و ترمینال را مجدداً باز کنید.

ایراد **Claude پس از پچ باز نمی‌شود (ویندوز):** اسکریپت را دوباره اجرا کنید، گزینه **۲ (بازگشت)** و سپس **۱ (نصب)** را انتخاب کنید.

ایراد **`Import-Module … AuditToString`:** از **PowerShell 7 (`pwsh`)** استفاده کرده‌اید که با پچ سازگار نیست. از **Windows PowerShell** (آبی‌رنگ) استفاده کنید: Win+R بزنید، `powershell` تایپ کنید و Enter بزنید.

ایراد **macOS — codesign با خطا مواجه شد:** مطمئن شوید Xcode Command Line Tools نصب است (`xcode-select --install`) و ترمینال به **Full Disk Access** در تنظیمات سیستم دسترسی دارد.

## ⚠️ سلب مسئولیت

> هشدار! [!CAUTION]
> این پچ فایل‌های باینری Claude Desktop (ویندوز) را تغییر می‌دهد یا یک کپی تغییریافته می‌سازد (macOS). نسخه ویندوز گواهینامه امضای Anthropic را با گواهینامه خودامضا جایگزین کرده و تأیید صحت برنامه را دور می‌زند.

با نصب، موارد زیر را می‌پذیرید:

۱. **استفاده با مسئولیت خودتان.** نویسندگان هیچ مسئولیتی در قبال آسیب، از دست دادن داده یا ناپایداری سیستم ندارند.
۲. **ممکن است با شرایط خدمات Anthropic همخوانی نداشته باشد.** پیشنهاد می‌شود آن‌ها را بررسی کنید.
۳. **به این مخزن اعتماد دارید.** اجرای کد با دسترسی مدیر خطرات خود را دارد — همیشه منبع را تأیید کنید.
۴. **موقتی است.** به‌روزرسانی‌های Claude پچ را پاک می‌کنند.
۵. **یک راه‌حل موقت است** تا زمانی که Anthropic پشتیبانی RTL بومی را اضافه کند.
